This is the first post on Minseok GitBlog.

The site is built with Hugo and published through GitHub Pages. That means posts can be written as Markdown files, committed to GitHub, and deployed automatically whenever the repository changes.

## What this blog is for

I will use this space for:

- Coding notes
- Git and GitHub practice
- Project logs
- Debugging records
- Ideas worth keeping

Small notes are welcome here. A post does not need to be perfect to be useful.
